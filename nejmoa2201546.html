FLAVOUR Trial

Fractional Flow Reserve or Intravascular Ultrasonography to Guide PCI

Koo BK, Hu X, Kang J, et al. — N Engl J Med 2022;387:779-89

2026-03-04

Background

Clinical Context

  • 관상동맥질환(CAD) 환자에서 PCI 결정 시 해부학적 평가와 생리학적 평가가 모두 중요
  • IVUS (Intravascular Ultrasonography)
    • 해부학적 정보 제공: lumen, vessel, plaque 상세 평가
    • IVUS-guided PCI → angiography-guided PCI 대비 임상 결과 개선 보고
  • FFR (Fractional Flow Reserve)
    • 생리학적 허혈 평가: 협착의 기능적 의미 판단
    • FFR-guided PCI → angiography-guided PCI 및 약물치료 대비 임상 이벤트 감소

Note

두 방법 모두 cardiac catheterization에서 가장 흔히 사용되는 보조 도구이나, head-to-head 비교 데이터가 부족

Background — Knowledge Gap

FFR vs IVUS: 역할의 차이

FFR IVUS
평가 대상 허혈 (physiology) 해부학 (anatomy)
주 용도 재혈관화 결정 PCI 계획 및 스텐트 최적화
기전 관상동맥 압력 비 측정 초음파 기반 혈관 내부 영상
장점 Ischemia-directed decision Plaque burden, lumen 정밀 측정
  • 실제 임상에서는 한 가지 도구만으로 PCI 결정 + 시술 가이드를 모두 수행하는 경우가 많음
  • 이 두 전략의 임상 결과 차이를 직접 비교한 RCT 필요

Study Objective

FLAVOUR Trial

Fractional Flow Reserve and Intravascular Ultrasound-Guided Intervention Strategy for Clinical Outcomes in Patients with Intermediate Stenosis


목적: 중등도 관상동맥 협착 환자에서 FFR-guided 전략이 IVUS-guided 전략 대비 비열등(noninferior)한지 평가


Major Protocol Revision

  • 최초 설계: FFR의 우월성(superiority) 검증 목적 (2016년 7월 시작)
  • 2017년 6월 프로토콜 변경 → 비열등성(noninferiority) 설계로 전환
    • 근거: IVUS-guided stenting의 추가적인 임상 이점을 보여주는 연구 결과들 발표
    • 정당성: FFR이 비열등하면서 스텐트 사용이 적다면 임상적으로 의미 있음

Study Design

Trial Overview

  • 디자인: Investigator-initiated, prospective, randomized, open-label, multinational
  • 센터: 한국 + 중국 18개 기관
  • 기간: 2016년 7월 ~ 2019년 8월 (등록), 24개월 추적
  • 대상: 4,355명 스크리닝 → 1,682명 무작위 배정
  • 배정: 1:1 (FFR군 838명 vs IVUS군 844명)
  • 층화 요인: 센터, 당뇨병 유무
  • 무작위화: Web-based concealed randomization
  • 후원: Boston Scientific (시험 설계, 데이터 분석, 논문 작성에 관여하지 않음)

Inclusion / Exclusion Criteria

Inclusion

  • 19세 이상
  • 허혈성 심질환 의심
  • 관상동맥 조영술상 de novo intermediate stenosis (40-70%)
  • Target vessel 직경 ≥ 2.5 mm (시각적 추정)

Exclusion

  • 비심장 동반질환으로 기대 여명 < 2년
  • Left main coronary artery 병변
  • Coronary artery bypass graft 병변
  • 출혈 위험 증가

Intervention — PCI 시행 기준

FFR Group (n=838)

PCI 시행 기준:

  • FFR ≤ 0.80
  • (관상동맥 혈류 ≥ 20% 감소)

Hyperemia 유발:

  • Adenosine IV (140 μg/kg/min) 또는
  • Adenosine triphosphate IV 또는
  • Intracoronary nicorandil (2 mg)

PCI 성공 기준:

  • Post-PCI FFR ≥ 0.88, 또는
  • Stent 전후 FFR 차이 < 0.05

IVUS Group (n=844)

PCI 시행 기준 (택 1):

  • MLA (Minimal Lumen Area) ≤ 3 mm², 또는
  • MLA 3-4 mm² + plaque burden > 70%


PCI 성공 기준:

  • Stent edge plaque burden ≤ 55%, 그리고
  • Minimal stent area ≥ 5.5 mm², 또는
  • Minimal stent area ≥ distal reference lumen area

Outcomes

Primary Outcome

  • 24개월 시점 복합 종말점:
    • Death (전체 사망)
    • Myocardial Infarction (심근경색)
    • Any Revascularization (모든 재혈관화)
  • Academic Research Consortium consensus 기준

Key Secondary Outcomes

  • Primary outcome 개별 구성 요소
  • 12개월 시점 복합 종말점
  • 환자당/혈관당 스텐트 수
  • Stroke
  • Seattle Angina Questionnaire (SAQ) — 환자 보고 결과
    • 5개 도메인, 0-100점 (낮을수록 증상 심함)

Statistical Analysis

Noninferiority Design

  • 가정: 24개월 primary outcome 발생률 — FFR군 10%, IVUS군 12%
  • 비열등성 마진: 2.5 percentage points
  • 검정력: 90%
  • 표본 수: 1,700명
  • Type I error: 일측 5% (의료기기 평가 기준)
    • 일측 2.5% (보수적 기준) 결과도 함께 보고

Analysis Methods

  • Cox proportional-hazards model (trial site, 당뇨병 유무 = random effect)
  • Kaplan-Meier method로 time-to-event 분석
  • ITT + per-protocol 분석 모두 수행
  • Proportional-hazards 가정: Schoenfeld residuals 검정
  • 다중비교 보정 없음 (secondary outcomes)

Results — Additional Analyses

Per-Protocol Analysis

  • FFR군 8.2% vs IVUS군 8.7%
  • 절대 차이: -0.5 pp (일측 97.5% CI 상한: 2.3 pp)
  • P = 0.02 for noninferiority → ITT 결과와 일관

Subgroup Analysis

  • 당뇨병 유무에 따른 층화 분석: 양 군 유사한 결과

Nicorandil 사용 환자 제외 Post Hoc Analysis

  • 프로토콜상 adenosine 사용이지만 288명에서 nicorandil 사용
  • 이들 제외 후에도 ITT 결과와 일관

PCI 성공률

  • 전체: 52.9% (사전 정의 기준)
  • FFR군 50.1% vs IVUS군 54.8% (유사)

Seattle Angina Questionnaire

  • 5개 도메인 모두 baseline 및 추적 시 양 군 유사

Discussion — Key Points

주요 해석

  1. FFR-guided 전략의 비열등성 확인
    • IVUS군 대비 PCI를 20.9% 적게 시행하면서도 24개월 임상 결과 동등
  2. 의료 자원 절감 효과
    • FFR군: 스텐트 수 감소, 스텐트 총 길이 감소
    • 이중 항혈소판 치료 빈도 감소
  3. 증상 측면에서도 동등
    • SAQ 점수: 양 군 유사 → 협착 정도와 증상이 항상 일치하지 않음
  4. 기존 연구와의 맥락
    • 최근 연구들: FFR-guided 전략의 우월성이 기대만큼 뚜렷하지 않음
    • IVUS-guided PCI: hard endpoints (사망, MI) 감소에 유리한 경향
    • Network meta-analysis: IVUS > angiography, FFR은 trend만

Limitations

해석 시 주의사항

  1. 저위험 환자군
    • 평균 SYNTAX score < 10 (낮은 해부학적 복잡성)
    • 고위험 환자에 일반화 제한
  2. 개방형 디자인 (Open-label)
    • 시술자가 배정군을 인지 → 추적 중 재혈관화 결정에 영향 가능
  3. PCI 기준에 따른 차이
    • 재혈관화 빈도는 PCI 기준 및 baseline 특성에 따라 달라질 수 있음
  4. 다른 기법과 비교 없음
    • iFR, QFR, OCT, NIRS-IVUS 등은 평가되지 않음
  5. Plaque vulnerability 미반영
    • PCI 기준에 병변 수준 hemodynamic significance, plaque 취약성 미포함
  6. 인종 제한
    • 아시아인(한국, 중국)만 포함 → 다른 인종에 일반화 주의

Limitations — Protocol Change

시험 설계 변경에 대한 고려

Superiority → Noninferiority 전환 (2017년 6월)

  • 시험 시작 후 약 1년 만에 프로토콜 변경
  • 정당화: IVUS-guided stenting의 추가 이점을 보여주는 연구 발표
  • Database lock: 2022년 1월 28일 (데이터 열람 전 변경)
  • 비열등성 마진 2.5 pp의 적절성에 대한 논란 가능
  • Type I error 일측 5% (일반적 2.5%보다 관대)
    • 단, 일측 2.5% 기준으로도 비열등성 충족 (CI 상한 2.2 pp)

Clinical Implications

임상적 시사점

FFR-guided 전략의 장점

  • 임상 결과 동등 (noninferiority)
  • PCI 시행률 감소 (44% vs 65%)
  • 스텐트 사용 감소
  • 이중 항혈소판제 사용 감소
  • 의료 비용 절감 가능

IVUS-guided 전략의 장점

  • 해부학적 정보 제공
  • Stent 최적화 가능
  • Hard endpoints 감소 경향 (기존 연구 기반)
  • PCI 시행 시 정밀한 계획 가능


“어떤 도구를 선택하느냐”보다 “각 도구의 강점을 적절히 활용하느냐”가 중요

— 실제 임상에서는 FFR + IVUS 병용이 최적일 수 있음

Conclusion

결론


중등도 관상동맥 협착(40-70%) 환자에서:

  • FFR-guided PCI는 IVUS-guided PCI 대비 비열등 (24개월 MACE 기준)
  • FFR군은 PCI를 44.4%에서만 시행 (IVUS군 65.3%)
  • 스텐트 사용 감소 + 유사한 임상 결과
  • 환자 보고 증상 (SAQ)도 양 군 유사


Future Directions

  • 고위험 환자에서의 검증 필요
  • FFR + IVUS 병용 전략 연구
  • OCT, iFR, QFR 등 다른 기법과의 비교
  • 비아시아 인종 포함 연구
  • Plaque vulnerability 반영한 PCI 기준 수립
